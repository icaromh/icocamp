## 💡 Como rodar os exercícios

Os exercícios usam `fetch`, que está disponível no browser e no Node.js a partir da versão 18.

**No browser**: Copie o código no console do DevTools (F12) e rode direto.

**No Node.js (v18+)**:

```bash
node 01-fetch-basico.js
```
