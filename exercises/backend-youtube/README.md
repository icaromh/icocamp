# 🎥 Exercícios: Backend com Node.js e Express (Tema: YouTube)

Bem-vindo(a) aos exercícios práticos de fixação para Node.js e Express! Neste conjunto, você vai construir a API de um Canal do YouTube do zero.

| Exercício | Descrição | Nível |
| --- | --- | --- |
| [01. Hello YouTube](./01-hello-youtube.md) | Criar servidor básico Express | 🟢 |
| [02. Listar Vídeos](./02-listar-videos.md) | Rota GET para array estático | 🟢 |
| [03. Detalhes do Vídeo](./03-detalhes-video.md) | Rota GET com Route Params | 🟡 |
| [04. Buscar Vídeos](./04-buscar-videos.md) | Rota GET com Query Params | 🟡 |
| [05. Publicar Vídeo](./05-publicar-video.md) | Rota POST com `req.body` | 🟡 |
| [06. Validar Publicação](./06-validar-publicacao.md) | Middleware de validação | 🔴 |
| [07. Editar Título](./07-editar-titulo.md) | Rota PUT para atualizar dados | 🟡 |
| [08. Dar Like](./08-dar-like.md) | Rota PATCH para atualizar likes | 🟡 |
| [09. Excluir Vídeo](./09-excluir-video.md) | Rota DELETE | 🟡 |
| [10. Logger Estatísticas](./10-logger-estatisticas.md) | Middleware Global de Logger | 🔴 |

> 💡 **Como fazer?**
> Tente implementar o código inicial em um arquivo separado, rode com `node seu-arquivo.js` e teste no seu navegador ou via Hoppscotch/Insomnia. Só olhe a pasta `solutions/` depois que tentar bastante!
