# 🧪 Prática Guiada — Banco de Dados (SQLite)

Esta pasta contém exercícios progressivos para fixação da linguagem SQL e da integração com Node.js.

> **Regra de Ouro:** Não decore SQL. Entenda a lógica por trás de cada comando.

---

## 📋 Lista de Exercícios

Recomendamos que você faça estes exercícios em um arquivo chamado `schema.sql` usando a extensão **SQLite Viewer** ou no **DBeaver Lite**.
Depois, para cada bloco, crie um arquivo Javascript (ex: `exercicios.js`) e implemente as queries usando `better-sqlite3`.

| Nível | Arquivo | Descrição |
| :---: | :--- | :--- |
| 🟢 | [01-crud-basico.md](./01-crud-basico.md) | Criar tabelas simples, INSERT, SELECT com WHERE e funções de agregação (SUM, COUNT). |
| 🟡 | [02-relacoes-e-join.md](./02-relacoes-e-join.md) | Criar tabelas com chaves estrangeiras (Foreign Keys) e buscar dados relacionados usando JOIN. |
| 🔴 | [03-blog-completo.md](./03-blog-completo.md) | Modelagem completa de um blog, cruzamento de dados complexos e estatísticas agrupadas (GROUP BY). |
