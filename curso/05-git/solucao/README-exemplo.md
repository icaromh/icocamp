# Minha Estante Virtual

Aplicação web para organizar livros lidos e não lidos, com persistência de dados no navegador.

## Funcionalidades

- 📚 Cadastro de livros com título, autor e status de leitura
- 📋 Listagem de todos os livros da estante
- 💾 Persistência automática via LocalStorage (os livros não somem ao recarregar)
- 🔍 Busca de livros por título em tempo real
- ⭐ Marcar livros como favoritos

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Como rodar localmente

Não precisa instalar nada! Basta:

1. Clonar o repositório:
```bash
git clone https://github.com/SEU_USUARIO/estante-virtual.git
```

2. Abrir o arquivo `index.html` no navegador

## Estrutura do projeto

```
estante-virtual/
├── index.html    # Estrutura da página
├── style.css     # Estilos e layout
├── app.js        # Lógica da aplicação
├── .gitignore    # Arquivos ignorados pelo Git
└── README.md     # Este arquivo
```

## Histórico de desenvolvimento

Este projeto foi desenvolvido ao longo do bootcamp IcoCamp, evoluindo a cada semana:

- **Semana 1**: Lógica de dados em JavaScript puro
- **Semana 2**: Estrutura HTML da página
- **Semana 3**: Estilos e layout com CSS
- **Semana 4**: Interatividade com DOM e LocalStorage
- **Semana 5**: Versionamento com Git e publicação no GitHub
