# Semana 01: Resumo Fundamentos de Programação e Algoritmia

**Pensamento computacional**

- Capacidade de analisar um problema e traduzi-lo em passos lógicos e executáveis por um computador.

**Decomposição de problemas**

- Quebrar um problema grande em partes menores e mais simples de resolver, tratando uma etapa por vez.

**Variáveis e tipos primitivos**

- Uso de variáveis para armazenar dados. Tipos básicos como Number, String, Boolean, null e undefined.

**Objetos e arrays**

- Estruturas para agrupar dados relacionados.
- Arrays organizam listas ordenadas; objetos organizam dados por chave/valor.

**Controle de fluxo (if / else)**

- Tomada de decisão no código com base em condições verdadeiras ou falsas.

**Laços (for, while, for...of)**

- Estruturas para repetir um bloco de código enquanto uma condição for atendida ou para percorrer coleções.

**Funções (declaração, parâmetros, retorno)**

- Blocos reutilizáveis de código que recebem dados de entrada (parâmetros) e produzem um resultado (return).

**Algoritmos simples (contagem, busca, agregação)**

- Soluções passo a passo para problemas comuns, como contar itens, encontrar um valor específico ou somar dados.

**Métodos de array: map, filter, reduce**

- map: transforma cada item do array
- filter: seleciona itens com base em uma condição
- reduce: reduz o array a um único valor (soma, contagem, agrupamento)

**Imutabilidade vs mutabilidade**

- Entender quando um dado é modificado diretamente (mutável) ou quando se cria uma nova versão sem alterar o original (imutável).

**Leitura e escrita de dados em estruturas JS**

- Acesso e modificação de valores em objetos e arrays usando índices, chaves e operadores (. e []).
