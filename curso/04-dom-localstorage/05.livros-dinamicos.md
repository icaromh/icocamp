📘 **Módulo: Renderização Dinâmica da Estante**

🎯 **O que vamos aprender?**
O objetivo é fazer com que o `BookSelf` mostre automaticamente todos os livros que você já cadastrou. Vamos aprender a ler a lista do `localStorage`, percorrer cada obra e usar as **Template Strings** para criar o HTML de cada "card" de livro, injetando-os na página sem precisar escrever código manual para cada um.

💡 **Teoria Aplicada**

### O Ciclo da Renderização Literária

Para exibir seus livros, seguimos um roteiro de três atos:

1. **Extração:** Buscamos a string no `localStorage` e a transformamos de volta em um objeto (ou array) de JavaScript usando `JSON.parse()`. É como tirar um livro da caixa e abri-lo para ler.
2. **Mapeamento:** Usamos um loop (como o `.forEach`) para visitar cada livro da sua lista.
3. **Injeção:** Para cada livro, criamos um "template" (molde) de HTML e o inserimos dentro de um container (geralmente uma `div` ou `section`) usando o `innerHTML`.

```javascript
// 1. Buscamos a 'coleção' (array de livros)
const colecaoSalva = JSON.parse(localStorage.getItem('minha_estante')) || [];

// 2. Selecionamos onde os livros vão aparecer
const vitrine = document.querySelector('#vitrine-livros');

// 3. Limpamos a vitrine antes de renderizar (para não duplicar)
vitrine.innerHTML = "";

// 4. Renderizamos cada obra
colecaoSalva.forEach((livro) => {
    vitrine.innerHTML += `
        <div class="card-livro">
            <h3>${livro.titulo}</h3>
            <p>Autor: ${livro.autor}</p>
        </div>
    `;
});
```

Imagine que sua aba de "Vídeos" no YouTube fosse estática: você teria que programar um novo bloco de código toda vez que subisse uma resenha. Seria exaustivo! Graças à **renderização dinâmica**, o site apenas pergunta ao banco de dados: *"O que temos hoje?"* e desenha os quadradinhos na tela. No `BookSelf`, faremos o mesmo: seu HTML será o palco, e os dados do `localStorage` serão os atores que entram em cena conforme você os cadastra.

✍️ **Exercícios de Fixação**

### 1. Desafio Rápido: O Tradutor de JSON

No seu console do navegador ou script, pratique a conversão. Imagine que você recebeu estes dados brutos de uma API literária:
`const dadosBrutos = '{"titulo": "O Morro dos Ventos Uivantes", "paginas": 368}';`

* Converta `dadosBrutos` para um objeto JavaScript chamado `livroObjeto`.
* Exiba no console apenas o número de páginas usando `livroObjeto.paginas`.

### 2. Dando Vida à Estante: A Vitrine Automática

Vamos atualizar sua função `init()` ou criar uma função dedicada chamada `renderizarEstante()`.

**O Desafio:**

1. Crie uma lista (array) de objetos no seu `localStorage` sob a chave `'meus_livros'`. (Dica: você pode cadastrar manualmente uns 3 livros para testar).
2. No seu arquivo `app.js`, crie a função `renderizarEstante()`.
3. Dentro dela:
    * Capture o elemento HTML onde os livros devem aparecer (ex: `<main id="lista-livros">`).
    * Recupere os livros do `localStorage`. **Atenção:** Se o `localStorage` estiver vazio, garanta que seu código use um array vazio `[]` para não quebrar.
    * Use um loop para criar um card para cada livro contendo o título e o autor.
    * Use `innerHTML` para "imprimir" esses cards na tela.

### 3. O Toque da Resenhista (Estilização Condicional)

Melhore a função de renderização: se a lista de livros estiver vazia, em vez de deixar a tela branca, exiba uma mensagem carinhosa: *"Sua estante ainda está esperando a primeira leitura. Que tal cadastrar um livro agora?"*
